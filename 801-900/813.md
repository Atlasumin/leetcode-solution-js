### 813. 最大平均值和的分组

难度：Middle

相关话题：`动态规划`

我们将给定的数组 `A` 分成 `K` 个相邻的非空子数组 ，我们的分数由每个子数组内的平均值的总和构成。计算我们所能得到的最大分数是多少。



注意我们必须使用 A 数组中的每一个数进行分组，并且分数不一定需要是整数。





```

示例:


输入:

 
A = [9,1,2,3,9]
K = 3
输出:

 20
解释:

 
A 的最优分组是[9], [1, 2, 3], [9]. 得到的分数是 9 + (1 + 2 + 3) / 3 + 9 = 20.
我们也可以把 A 分成[9, 1], [2], [3, 9].
这样的分组得到的分数为 5 + 2 + 6 = 13, 但不是最大值.

```

 **说明: ** 





*  `1 &lt;= A.length &lt;= 100` .

*  `1 &lt;= A[i] &lt;= 10000` .

*  `1 &lt;= K &lt;= A.length` .

* 答案误差在 `10^-6` 内被视为是正确的。






```
/**
 * @param {number[]} A
 * @param {number} K
 * @return {number}
 */
var largestSumOfAverages = function(A, K) {
        let N = A.length;
        let memo = []
        for(let i=0;i&lt;N+1;i++){
            memo[i]=Array(N+1).fill(0)
        }
        let cur = 0;
        for (let i = 0; i &lt; N; ++i) {
            cur += A[i];
            memo[i + 1][1] = cur / (i + 1);
        }
    
        return search(N, K, A, memo);
    
    function search( n,  k,  A, memo) {
        // console.log(k)
        if (memo[n][k] &gt; 0) return memo[n][k];
        if (n &lt; k) return 0;
        let cur = 0;
        for (let i = n - 1; i &gt; 0; --i) {
            cur += A[i];
            memo[n][k] = Math.max(memo[n][k], search(i, k - 1, A, memo) + cur / (n - i));
        }
        // console.log(memo)
        return memo[n][k];
};
    }
```
