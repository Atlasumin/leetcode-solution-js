### 125. 验证回文串

难度：Easy

相关话题：`双指针`、`字符串`

给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。



 **说明：** 本题中，我们将空字符串定义为有效的回文串。



 **示例 1:** 





```
输入:

 "A man, a plan, a canal: Panama"
输出:

 true

```

 **示例 2:** 





```
输入:

 "race a car"
输出:

 false

```


```
/**
 * @param {string} s
 * @return {boolean}
 */
var isPalindrome = function(s) {
  if(s==="")return true
  s = s.toLowerCase().replace(/[^a-z0-9]/g, "");

  let l=0,r=s.length-1
  while(l&lt;r){
    if(s[l]!==s[r])return false
    l++;r--
  }
  return true
};



```
