### 301. 删除无效的括号

难度：Hard

相关话题：`深度优先搜索`、`广度优先搜索`

删除最小数量的无效括号，使得输入的字符串有效，返回所有可能的结果。



**说明:**  输入可能包含了除 `(` 和 `)` 以外的字符。



**示例 1:** 



```
输入: "()())()"
输出: ["()()()", "(())()"]
```


**示例 2:** 



```
输入: "(a)())()"
输出: ["(a)()()", "(a())()"]
```


**示例 3:** 



```
输入: ")("
输出:[""]
```
