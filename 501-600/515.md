### 515. 在每个树行中找最大值

难度：Middle

相关话题：`树`、`深度优先搜索`、`广度优先搜索`

您需要在二叉树的每一行中找到最大的值。



**示例：** 



```

输入:

 

          1
         / \
        3   2
       / \   \  
      5   3   9 

输出:

 [1, 3, 9]
```

```
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var largestValues = function(root) {
  if(!root)return []
  let ans=[]
  let arr=[root]
  while(arr.length>0){
    let len=arr.length
    let maxV=-Infinity
    for(let i=0;i<len;i++){
      let node=arr.shift()
      maxV=Math.max(maxV,node.val)
      if(node.left)arr.push(node.left)
      if(node.right)arr.push(node.right)
    }
    ans.push(maxV)
  }
  return ans
};
```
