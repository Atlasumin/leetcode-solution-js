### 917. 仅仅反转字母

难度：Easy

相关话题：`字符串`

给定一个字符串 `S` ，返回&ldquo;反转后的&rdquo;字符串，其中不是字母的字符都保留在原地，而所有字母的位置发生反转。












**示例 1：** 



```
输入：

"ab-cd"
输出：

"dc-ba"
```


**示例 2：** 



```
输入：

"a-bC-dEf-ghIj"
输出：

"j-Ih-gfE-dCba"
```


**示例 3：** 



```
输入：

"Test1ng-Leet=code-Q!"
输出：

"Qedo1ct-eeLg=ntse-T!"
```






**提示：** 




1.  `S.length <= 100` 

2.  `33 <= S[i].ASCIIcode <= 122` 

3.  `S`  中不包含 `\`  or  `"` 




```
/**
 * @param {string} S
 * @return {string}
 */
var reverseOnlyLetters = function(S) {
  let rs=''
  for(let i=S.length-1;i>=0;i--){
    if(/[a-zA-Z]/.test(S[i]))rs+=S[i]
  }
  let result='',j=0
  for(let i=0;i<S.length;i++){
    if(/[a-zA-Z]/.test(S[i]))result+=rs[j++]
    else result+=S[i]
  }
  return result
};
```
