### 41. 缺失的第一个正数

难度：Hard

相关话题：`数组`

给定一个未排序的整数数组，找出其中没有出现的最小的正整数。



 **示例1:** 





```
输入: [1,2,0]
输出: 3

```

 **示例2:** 





```
输入: [3,4,-1,1]
输出: 2

```

 **示例3:** 





```
输入: [7,8,9,11,12]
输出: 1

```

 **说明:** 



你的算法的时间复杂度应为O( *n* )，并且只能使用常数级别的空间。




```
/**
 * @param {number[]} nums
 * @return {number}
 */
var firstMissingPositive = function(A) {
  let n=A.length
  for(let i = 0; i &lt; n; ++ i)
    while(A[i] &gt; 0 &amp;&amp; A[i] &lt;= n &amp;&amp; A[A[i] - 1] != A[i])
    swap(i, A[i] - 1);

  for(let i = 0; i &lt; n; ++ i)
    if(A[i] != i + 1)
      return i + 1;

  function swap(i,j){
    let t=A[i]
    A[i]=A[j]
    A[j]=t
  }
  return n + 1;
};



```
