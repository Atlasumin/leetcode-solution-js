### 14. 最长公共前缀

难度：Easy

相关话题：`字符串`

编写一个函数来查找字符串数组中的最长公共前缀。



如果不存在公共前缀，返回空字符串 `""` 。



 **示例1:** 





```
输入: 

["flower","flow","flight"]
输出:

 "fl"

```

 **示例2:** 





```
输入: 

["dog","racecar","car"]
输出:

 ""
解释:

 输入不存在公共前缀。

```

 **说明:** 



所有输入只包含小写字母 `a-z` 。




```
/**
 * @param {string[]} strs
 * @return {string}
 */
var longestCommonPrefix = function(strs) {
  if(strs.length===0)return ''
  let comPre=strs[0]
  for(let i=1;i&lt;strs.length;i++){
    let j=0
    for(;j&lt;strs[i].length;j++){
      if(strs[i][j]!==comPre[j])break
    }
    if(j===0)return ''
    comPre=comPre.substring(0,j)
  }
  return comPre
};



```
