### 67. 二进制求和

难度：Easy

相关话题：`数学`、`字符串`

给定两个二进制字符串，返回他们的和（用二进制表示）。



输入为**非空** 字符串且只包含数字 `1` 和 `0` 。



**示例1:** 



```
输入:

 a = "11", b = "1"
输出:

 "100"
```


**示例2:** 



```
输入:

 a = "1010", b = "1011"
输出:

 "10101"
```

```
/**
 * @param {string} a
 * @param {string} b
 * @return {string}
 */
var addBinary = function(a, b) {
  let res='',carry=0
  let i=a.length-1,j=b.length-1
  while(i>=0 || j>=0){
    let sum
    if(i===-1){
      sum=+b[j--]+carry
    }else if(j===-1){
      sum=+a[i--]+carry
    }else{
      sum=+a[i--] + +b[j--] +carry
    }
    carry=Math.floor(sum/2)
    sum %=2
    res=sum+''+res
  }
  if(carry>0)return carry+''+res
  return res
};
```
